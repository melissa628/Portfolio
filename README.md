# My Professional Portfolio

A personal website showcasing my skills, projects, and journey as a Software Engineering student.

##  Features
* **Modern UI:** Built with a clean, responsive design.
* **Dark Mode:** Support for user-preferred color schemes.
* **Bilingual Support:** Easily switch between English and French.
* **Responsive Layout:** Optimized for mobile, tablet, and desktop views using CSS Flexbox/Grid.

##  Built With
* **HTML:** Semantic structure for better SEO and accessibility.
* **CSS:** Custom styling with modern hover effects and transitions.
* **JavaScrip:** Logic for the theme toggler and language switching.

##  Project Structure
* `index.html` - The main entry point.
* `portfolio.css` - Custom styles and layouts.
* `portfolio.js` - Interactive elements and logic.
